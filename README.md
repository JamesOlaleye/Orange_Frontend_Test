# My Products App

## How to run the project

1. Clone the repository:

   <!-- git clone https://github.com/yourusername/my-products-app.git -->

2. Navigate to the project folder:

    cd frontend-test

3. Install dependencies: 

    yarn

4. Run the development server

    yarn dev

5. Open your browser and go to http://localhost:5173/

# PROJECT STRUCTURE BELOW:

--- src/components: This Contains reusable React components.
--- src/pages: Contains the page components for different views.
--- src/App.js: Main application component with React Router setup.
--- src/index.js: Is the Entry point for the React application.
